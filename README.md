```groovy
compile 'com.github.hajiyevelnur92:intentanimation:1.0'
```  

## **Usage** ##
#### **Code**
```java
customType(MainActivity.this,mystring);
//mystring has to be one of bellow string
//left-to-right
//right-to-left
//bottom-to-up
up-to-bottom
fadein-to-fadeout
rotateout-to-rotatein
```
